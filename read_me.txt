		Spreadsheet evaluator by Tadas Vasiliauskas

Script needs Python 3.10 or later version (where is no SWITCH (match case) function in older versions)

Script needs "requests" module, to install it run "pip install requests" in your terminal

To run tests "pytest" module is needed, to install it run "pip install pytest"

To run tests use "pytest" in your terminal

All the functions in the code are fully tested, to see report: coverage_re\index.html open spreadsheet_evaluator.py 

To generate new tests coverage report run "pip install pytest-cov" and run "python -m pytest --cov --cov-report=html:coverage_re"
